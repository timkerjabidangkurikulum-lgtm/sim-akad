# Aplikasi Perencanaan Kokurikuler

Aplikasi web untuk membuat perencanaan kegiatan kokurikuler dengan pendekatan **Deep Learning** dan **Kurikulum Berbasis Cinta (KBC)** di Madrasah Aliyah Negeri Banggai.

## Fitur

- ✅ Form input informasi madrasah (nama, fase, kelas, tahun ajaran)
- ✅ Form detail perencanaan dengan pendekatan Deep Learning dan KBC
- ✅ Preview real-time dokumen perencanaan
- ✅ Download dokumen sebagai file Microsoft Word (.doc)
- ✅ Desain responsif dengan gradient background
- ✅ Logo Kementerian Agama di header

## Cara Menggunakan

1. Buka aplikasi di browser
2. Isi informasi dasar madrasah (nama, fase, kelas, tahun ajaran)
3. Isi detail perencanaan:
   - Tema kokurikuler
   - Pendekatan Deep Learning dan KBC
   - Kegiatan yang akan dilakukan
   - Asesmen/penilaian
4. Klik "Generate Preview" untuk melihat hasil
5. Klik "Download Dokumen (Word)" untuk mengunduh file

## Teknologi

- HTML5
- CSS3 (Glassmorphism design)
- Vanilla JavaScript
- Client-side document generation

## Deployment

Aplikasi dapat di-deploy ke Vercel untuk akses online:

```bash
vercel deploy
```
